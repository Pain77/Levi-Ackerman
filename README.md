## INSTALLATION ON TERMUX
```bash
> pkg install nodejs && pkg install git
> git clone https://github.com/Pain77/Levi-Ackerman
> cd bitch-bot
> npm install terminal-kit
> bash install.sh
> npm start/node index.js
```
## INSTALLATION ON PC
```bash
> git clone https://github.com/Pain77/Levi-Ackerman
> cd bitch-bot
> npm install terminal-kit
> npm i
> npm start/node index.js
```
